# BQ76952EVM

High accuracy battery monitor and protector for 3-cell to 16-cell Li-ion or Li-polymer battery packs.

## References

[Product](https://www.ti.com/tool/BQ76952EVM)

[Software Dev Guide](https://www.ti.com/lit/an/sluaa11b/sluaa11b.pdf)

## Setup

Instead of using an MSPM0 LaunchPad, a Raspberry Pi 5 was used, with its external I²C pins connected to SDA and SCL. A 12 V DC power supply was applied to power the TI board.
